REST Webservices Beispiele mit Tomcat V8.5, Jax-RS 2.0 (Jersey 2.25)

http://localhost:8080/Jax-RS-Tomcat/rest/demo/hello -- Hello World, Plain Text, ohne Parameter

http://localhost:8080/Jax-RS-Tomcat/rest/demo/halloText?name=Andreas -- Hallo Name, Plain Text, mit String Parameter

http://localhost:8080/Jax-RS-Tomcat/rest/demo/sum/1/2 -- Hallo Name, Plain Text, mit String Parameter
